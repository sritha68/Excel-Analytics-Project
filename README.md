Tesla Social Media Analytics – Excel Project
End-to-End Marketing Analytics using Excel | Pivot Tables | Dashboard | KPI Insights

This project analyzes Tesla’s social media performance across campaigns, platforms, and content formats using Excel. It includes data cleaning, transformation, pivot tables, charts, dashboards, and strategic insights for decision-making.

Key Highlights:

1. Data cleaning & transformation

2. Pivot tables for KPI analysis

3. Engagement rate & ROI analysis

4. Hashtag performance evaluation

5. Content type effectiveness

6. Campaign-wise impressions, likes, clicks

7. Follower growth trends

8. Dashboard for executive view

9. Insights + recommendations

Project Structure:
1. Posts Dataset – Raw social media posts

2. Engagement Summary – Weekly metrics

3. Campaign Metadata – Campaign windows

4. Tasks 1–6 – Analytics for each KPI

5. Interactive Excel Dashboard

Techniques Used:
1. Pivot Tables

2. VLOOKUP / INDEX-MATCH

3. IF, DATE, TEXT functions

4. Calculated fields

5. Charts (Line, Pie, Donut, Clustered Column, Combo)

6. Conditional formatting

Final Dashboard:

A complete dashboard showing campaign performance, platform engagement, ROI, and follower growth.

Key Insights:
1. SustainabilityDrive campaign achieved the highest engagement & ROI

2. Instagram & Facebook delivered stronger engagement

3. Twitter & YouTube drove follower growth

4. Carousels & Videos were the best performing content types

5. Strong follower surge occurred during overlapping campaigns

6. Ad spend had weak correlation with follower growth

Recommendations:
1. Maintain a multi-platform strategy

2. Prioritize video, reels, carousels & stories

3. Post sustainability + innovation content

4. Optimize ad budget toward high-ROI campaigns

Tools Used:
1. Microsoft Excel
 
2. Pivot Tables

3. Charts & Dashboard

4. Power Query
